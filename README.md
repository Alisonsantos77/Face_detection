## Descrição

Este é um projeto simples que utiliza OpenCV e MediaPipe para detectar rostos em tempo real usando a webcam do seu computador.

## Requisitos

- Python 3.x
- OpenCV (`pip install opencv-python`)
- MediaPipe (`pip install mediapipe`)
